Hello, my name is Ingrid

Ingrid is a lightweight and fluid CSS layout system, who's main goal is to reduce the use of classes on individual units. Making it feel a bit less obtrusive and bit more fun to reflow for responsive layouts.

Ingrid is also meant to be an extendable system, easy to customize to your own needs.
